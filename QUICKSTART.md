# 快速启动指南

## 方法一：使用启动脚本（推荐）

**Windows用户**：
1. 双击运行 `start.bat`
2. 等待依赖安装和服务器启动
3. 在浏览器中打开 `index.html`

## 方法二：手动启动

### 步骤1：安装依赖
```bash
pip install -r requirements.txt
```

### 步骤2：启动后端
```bash
python backend.py
```

### 步骤3：打开前端
在浏览器中打开 `index.html` 文件

或者使用简单HTTP服务器：
```bash
# Python 3
python -m http.server 8080

# 然后访问 http://localhost:8080
```

## 使用流程

### 1️⃣ 输入研究方向
在首页输入框输入你的研究方向，例如：
- "机器学习在医疗诊断中的应用"
- "深度学习"
- "自然语言处理"

### 2️⃣ 浏览论文卡片
- 查看论文的标题、作者、摘要
- 系统会返回5篇相关论文

### 3️⃣ 选择论文
**左滑或点击 ❌**：跳过论文
**右滑或点击 ❤️**：保存到储存库

### 4️⃣ 查看结果
完成所有论文筛选后，可以：
- 查看已保存的论文列表
- 开始新的搜索

## 交互技巧

### 鼠标操作
- 拖拽卡片向左/右移动
- 松开鼠标完成滑动
- 点击底部按钮快速选择

### 触摸操作（移动设备）
- 用手指左右滑动卡片
- 滑动超过一半自动完成

### 键盘快捷键
- `Enter`：开始搜索
- 点击建议标签快速填充

## 常见问题

### Q: 后端启动失败？
A: 确保已安装Python 3.x和所有依赖：
```bash
pip install Flask Flask-CORS
```

### Q: 前端无法连接后端？
A: 检查：
1. 后端是否在运行（http://localhost:5000）
2. 浏览器控制台是否有CORS错误
3. 防火墙是否阻止了连接

### Q: API返回错误？
A: 检查：
1. API凭证是否正确
2. 网络连接是否正常
3. API配额是否用完

### Q: 卡片不能滑动？
A: 尝试：
1. 刷新页面
2. 使用Chrome或Firefox浏览器
3. 检查浏览器控制台错误
